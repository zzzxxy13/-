# 接口文档

后端地址：

```text
http://localhost:8080
```

登录后需要请求头：

```text
Authorization: Bearer user-用户ID-角色
```

## 认证

```text
POST /api/auth/register
POST /api/auth/login
```

请求：

```json
{
  "username": "alice",
  "password": "123456"
}
```

返回：

```json
{
  "id": 1,
  "username": "alice",
  "role": "USER",
  "token": "user-1-USER"
}
```

## 梦境记录

```text
POST /api/dreams
GET /api/dreams
GET /api/dreams/{dreamId}
PUT /api/dreams/{dreamId}
DELETE /api/dreams/{dreamId}
```

新增 / 修改请求：

```json
{
  "title": "考试和坠落",
  "content": "我梦见自己在学校考试，后来从楼梯上坠落。",
  "dreamDate": "2026-07-07",
  "moodScore": 2
}
```

## 梦境分析

```text
POST /api/dreams/{dreamId}/analyze
GET /api/dreams/{dreamId}/analysis
```

返回：

```json
{
  "dreamId": 1,
  "matchedKeywords": "学校,考试,坠落",
  "ruleBasedResult": "本次梦境命中了3个关键词...",
  "aiResult": "{\"atmosphere\":\"...\"}",
  "riskLevel": "LOW"
}
```

## 匿名社区

```text
POST /api/dreams/{dreamId}/share
GET /api/community/dreams
GET /api/community/dreams/{dreamId}
```

## 管理员

```text
GET /api/admin/community/dreams

```
