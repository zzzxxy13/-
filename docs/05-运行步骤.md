# 运行步骤

## 后端

```bash
cd D:\梦境馆\dream-archive-python-mvp\backend-python
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python init_db.py
uvicorn app.main:app --reload --port 8080
```

后端接口文档：

```text
http://localhost:8080/docs
```

## 前端

```bash
cd D:\梦境馆\dream-archive-python-mvp\frontend-vue
npm install
npm run dev
```

前端地址：

```text
http://localhost:5173
```
